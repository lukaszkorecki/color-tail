color-tail
==========

Like tail -f but with colors tags for each file


usage
=====

`color-tail /path/to/file1.log /path/to/another/file2.log`

![example](http://i.imgur.com/JmzAu7N.png)


installation
============

`go install https://github.com/lukaszkorecki/color-tail/`


current version
===============
